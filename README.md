# CS-study Rule

- 각자 15분 내외 설명하는 분량으로 주제를 선정

### 시간표 (월~금)

**1. 오전 10시 30분 - 11시 50분 : CS 발표**<br>
2. ~ 오후 1시 : 점심시간<br>
3. ~ 오후 6시 : PS 공부 & CS 복습<br>
4. ~ 오후 7시 : 저녁 시간<br>
5. ~ 코어타임 끝 : CS 공부<br>
